FastMath.Net
========

Fast Math is simple C# library for Fixed Point arithmetics based on int.

But seems to be it's much easier to use just float or even double than do anything special.
For 100000000 calculations.

| Type   | Time |
|--------|------|
| double | 54ms |
| float  | 56ms |
| int    | 43ms |
| Fast24 | 72ms |
